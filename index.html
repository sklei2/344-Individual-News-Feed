Ajax Activity

This activity is to show an understanding and proficiency in ajax by creating a webpage that will do four things. One, load some information from the server about me. Two, Be able to read JSON files from the server, both malformed and well formed. Three, read and display an RSS feed. Four, have it so we display random information that is continually added whenever the user scrolls to the bottom.
